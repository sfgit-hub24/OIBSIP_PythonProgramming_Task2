# OIBSIP Python Programming Internship
## Task 2: Random Password Generator

### Description
This project is a Random Password Generator developed using Python.  
The program generates strong and secure passwords based on user-defined length using a combination of uppercase letters, lowercase letters, digits, and special characters.

### Technologies Used
- Python
- Random Module
- String Module

### Features
- Generates strong and secure passwords
- User chooses the desired password length
- Uses letters, numbers, and special characters
- Simple and beginner-friendly implementation
